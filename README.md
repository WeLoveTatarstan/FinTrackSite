# FinTrackSite

## Версия alpha1.4

### Обновления конвертера валют

**Новые возможности:**
- ✅ **Поиск валют** - текстовый поиск по коду валюты с мгновенным результатом
- ✅ **Сортировка валют** - возможность сортировки по:
  - Коду валюты (А-Я, Я-А)
  - Курсу (по возрастанию, по убыванию)
- ✅ **Пагинация** - отображение валют по 5 штук на странице с навигацией
- ✅ **Динамическое управление** - все функции работают без перезагрузки страницы

**Технические улучшения:**
- Обновлен JavaScript для управления валютами
- Добавлены элементы управления в HTML шаблон
- Улучшен пользовательский интерфейс конвертера
